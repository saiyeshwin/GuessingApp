# Guessing App\
## Use Case 1 
Game Initialization – Random number generation, max attempts, hints setup, and rules display

## Use Case 2 
User Guess Submission – Accept input, validate, compare with target, track attempts, give feedback

## Use Case 3  
Hint Generation – Provide limited hints (even/odd, range, divisibility) after incorrect guesses

## Use Case 4  
Error Handling & Validation – Handle invalid input with exceptions, prevent crashes, show friendly messages

## Use Case 5  
Game Result Storage – Save player results (name, attempts, win/loss) in file or database for persistence

## Use Case 6  
Game Restart & Exit – Allow replay, reset variables, close resources, exit gracefully
